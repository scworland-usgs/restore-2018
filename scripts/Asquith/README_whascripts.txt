The scripts herein often refer to a DV.RData file. This is a large file (98MB) containing an R environment called DV. This DV holds daily values for the 1,379 streamgages. This file is close to the 100MB upper limit of file size for github.com. This file contains the same content as the Feather files under the /data directory at the top of this archive. 

Most of the content herein is dependent on the akqdecay package by Asquith written for this project.
